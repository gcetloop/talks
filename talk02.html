2. Setting up Git & Github.

Congratulations!!!! You have git installed in your system. Open your git bash and now is the time to set up git.

Set your name and email in your local git setting. To do so, use following command.

$ git config --global user.name "Your Name"
      # Example: git config --global user.name "Piyush Maurya"
$ git config --global user.email "Email"
      # Example: git config --global user.email "piyush.maurya@live.in"

Now you set up git in your local system, it is time to geerate a RSA key so that you can connect to your github repo using SSH without typing your username and password everytime.

$ git push origin master
