Talk 02: Introduction to Git & Github

1. Installing git and git-bash in windows.

Steps for Installing Git and Gitbash

Steps

  1. Go to https://git-for-windows.github.io/ and download and install the file.(File size=33.1 MB)
  2. Double click on downloaded file and Click on RUN when you prompted.
  3. Check git-bash is installed successfully in your Programs.

2. Setting up Git & Github.

Congratulations!!!! You have git installed in your system. Open your git bash and now is the time to set up git.

Set your name and email in your local git setting. To do so, use following command.

$ git config --global user.name "Your Name"
# Example: git config --global user.name "Piyush Maurya"
$ git config --global user.email "Email"
# Example: git config --global user.email "piyush.maurya@live.in"

Now you set up git in your local system, it is time to geerate a RSA key so that you can connect to your github repo using SSH without typing your username and password everytime.

$ ssh-keygen -t rsa -b 4096 -C "your_email@example.com"

This will start creating a ssh key using your email id.

Generating public/private rsa key pair.

Now your will be prompted to enter file name, just press ENTER button in this step.

Enter a file in which to save the key (/Users/you/.ssh/id_rsa): [Press enter]

You will be asked for passphrase, do not enter any passphrase.

Enter passphrase (empty for no passphrase): [Type a passphrase]
Enter same passphrase again: [Type passphrase again]

Start ssh-agent

$ eval "$(ssh-agent -s)"

Then, add your newly generated key to ssh-agent.

$ ssh-add ~/.ssh/id_rsa

Now login to your github account, goto settings >> SSH and GPG keys >> New SSH key >> Enter your SSH key and click ADD SSH key.

Your ssh key will be found in /.ssh/id_rsa_pub.

You are good to go. Start using github using SSH.
